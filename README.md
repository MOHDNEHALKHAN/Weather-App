# Weather App
It can give real time weather information of your location

## Tech Stack

**Client :** `Html, Css, Javascript`

**Server :** `Node.js`

**Api :** `Open Weather Api`

## Features
- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform
